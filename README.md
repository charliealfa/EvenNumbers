* Armstrong Numbers -> Girilen sayının digitlerinin basamak sayısı kadar kuvvetlerinin toplamının kendisine eşitliğini kontrol eden program
* Harmonic Serie -> Girilen sayıya kadar olan harmonik seriyi hesaplar
